Improvements Made:

  • ✅ Less rounded: Border-radius reduced from 4px → 2px
  • ✅ More spacious: Container height 28px → 32px, padding increased
  • ✅ Wider bar: Stats bar width 60px → 80px for better text display
  • ✅ Bigger text: Percentage text 10px → 11px, looks less cramped
  • ✅ Lower threshold: Text shows for segments >8% (was >10%)
  • ✅ Removed redundant total percentage - individual percentages are enough
  • ✅ Better gap spacing: 8px → 10px between elements
